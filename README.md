# NIKHEFProject2018

Thank you for your interest in this C++/ROOT framework analyse beam data. We developed this framework specifically for the NIKHEF Student Project from April to June 2018, but with the intention to develop it in such a way that it can be adepted for any other beam analysis setup.

## Acknowledgements
The NIKHEF Project 2018 was supervised by Daniel Hynds, who codeveloped [Corryvreckan](https://gitlab.cern.ch/dhynds/corryvreckan) and recommended us to follow its structure.
